Sales Analysis Report
       Developed a comprehensive four-page interactive Power BI dashboard to analyze sales data across categories, regions, time periods, and customer age groups The dashboard provided visual insights into trends, profitability, and business metrics to support data-driven decision making.

Key Features :
•	Cleaned and prepared data for dashboard creation.
•	Designed multiple visuals and pages: line charts, bar charts, pie charts, scatter plots, map charts.
•	Created summary cards: Total Sales, Sum of Profit, Count of States, Count of Orders.
•	Implemented filters (e.g. Year, region, category to make the dashboard interactive used DAX measures in Power BI for calculations and aggregations integrated multiple tables customer data, product, transactions into a unified model.)
